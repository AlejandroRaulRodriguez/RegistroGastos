# Registro de Gastos

Aplicación en React para registrar, visualizar, editar y eliminar gastos personales usando `json-server` como backend REST y `axios` para consumir la API.

## Scripts

- `npm run dev`: inicia el frontend con Vite.
- `npm run server`: levanta `json-server` en `http://localhost:3001` usando `db.json`.
- `npm run build`: genera la versión de producción.
- `npm run lint`: ejecuta ESLint.

## Flujo de uso

1. Ejecutá `npm run server` en una terminal.
2. Ejecutá `npm run dev` en otra terminal.
3. Usá el formulario para crear gastos, o los botones de la lista para editarlos y eliminarlos.
